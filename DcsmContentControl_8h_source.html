277  /*
278  * @brief Trigger renderer to test if given pick event with coordinates intersects with any instances
279  * of ramses::PickableObject contained in given content's scene. If so, the intersected PickableObjects are
280  * reported to the ramses::DcsmContentControl (see ramses::IDcsmContentControlEventHandler::objectsPicked) using their user IDs
281  * given at creation time (see ramses::Scene::createPickableObject).
282  *
283  * @details \section Coordinates
284  * Coordinates normalized to range <-1, 1> where (-1, -1) is bottom left corner of the buffer where content is mapped to
285  * and (1, 1) is top right corner.
286  * If the content to test is rendered directly to framebuffer then display size should be used,
287  * i.e. (-1, -1) is bottom left corner of the display and (1, 1) top right corner of display.
288  * If the content is mapped to an offscreen buffer and rendered as a texture mapped
289  * on a mesh in another content, the given coordinates need to be mapped to the offscreen buffer
290  * dimensions in the same way.
291  * For example if the content's offscreen buffer is mapped on a 2D quad placed somewhere on screen
292  * then the coordinates provided need to be within the region of the 2D quad, i.e. (-1, -1) at bottom left corner of the quad and (1, 1) at top right corner.
293  *
294  * @param contentID Content to check for intersected PickableObjects.
295  * @param bufferNormalizedCoordX Normalized X pick coordinate within buffer size (see \ref Coordinates).
296  * @param bufferNormalizedCoordY Normalized Y pick coordinate within buffer size (see \ref Coordinates).
297  * @return StatusOK for success, otherwise the returned status can be used
298  * to resolve error message using getStatusMessage().
299  */
300  status_t handlePickEvent(ContentID contentID, float bufferNormalizedCoordX, float bufferNormalizedCoordY);
