class  ramses::PerspectiveCamera
 The PerspectiveCamera is a local camera which defines a perspective view into the scene. Frustum planes can be set explicitly via setFrustum of base class LocalCamera. Alternatively a symmetric frustum can be set using field of view and aspect ratio. The top and bottom planes are defined by the "vertical field of view" parameter, which specifies the angle between top and bottom plane. More...
