class  ramses::LocalCamera
 The LocalCamera base class is part of a scene and defines a view into the scene defined by the client application. The "locality" means that the client has full control over the camera projection and can set viewport and frustum parameters. The LocalCamera can be used for remote scenes, but it will override the renderer projection parameters and viewport. It is recommended to use a RemoteCamera for scenes which are supposed to be distributed. More...
