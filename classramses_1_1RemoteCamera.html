Detailed Description

The RemoteCamera is part of a scene and defines a view into the scene defined by the (possibly remote) renderer display where this scene is mapped. The viewport is taken from the display window, and projection parameters are set on the display config. A client can not set/override these values when using a RemoteCamera.
